# 审计日志

## 审计日志思路
![](../doc/审计日志思路图.png)

数据修改日志插件对各种mapper的增删改进行监控，且仅需升级最新版本的Druid库并作少量更改即可支持大部分的数据库（目前版本支持MySQL）。
本插件通过拦截器记录Mybatis下所有Mapper类在进行增删改动作时所影响数据的 oldValue（修改前，插入前，删除前），newValue（修改后，插入后，删除后），
